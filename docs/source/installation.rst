Installation
============

Basic Installation
-----------------
LatenPy can be installed using pip:

.. code-block:: bash

   pip install freezedry

Development Installation
-----------------------
To install LatenPy for development:

1. Clone the repository:

   .. code-block:: bash

      git clone https://github.com/landoskape/freezedry.git
      cd freezedry

2. Install in development mode with test dependencies:

   .. code-block:: bash

      pip install -e ".[test]"
